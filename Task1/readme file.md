# Task 1: Responsive Webpage Development 
The objective of Task 1 is to build a **simple, clean, and fully responsive webpage** using **HTML, CSS, and JavaScript**.

## Features 

# Fully Responsive Layout  
The webpage adapts smoothly to:
- Desktop screens  
- Tablets  
- Mobile devices  

#  Sections Included
- **Header** with navigation menu  
- **Mobile navigation menu toggle**  
- **Hero section** with title & call-to-action  
- **Features section** (3 responsive cards)  
- **Footer**  

#  Technologies Used
- **HTML5**
- **CSS3**
  - Flexbox  
  - Media queries  
  - JavaScript
  - Mobile menu toggle function
