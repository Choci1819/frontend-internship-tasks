# React Single-Page Application (SPA)

This project is a React-based Single-Page Application (SPA) that demonstrates dynamic data fetching and smooth navigation.

## Features

- **Dynamic Data Fetching**: Fetches data from an API and displays it dynamically.
- **Smooth Navigation**: Implements navigation between pages using React Router.
- **Interactive UI Updates**: Styled for a clean and modern user experience.

## Project Structure

- `src/App.js`: Main component that fetches and displays data.
- `src/App.css`: Styles for the application.
- `src/index.js`: Entry point for the React application.

## How to Run

1. Navigate to the project directory:
   ```bash
   cd spa-task3
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm start
   ```
4. Open your browser and go to [http://localhost:3000](http://localhost:3000).

## API Used

- [JSONPlaceholder](https://jsonplaceholder.typicode.com/posts): A free fake API for testing and prototyping.

## Customization

- Update `src/App.css` to modify the UI.
- Change the API endpoint in `src/App.js` to fetch different data.
