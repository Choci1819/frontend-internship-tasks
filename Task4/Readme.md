# Task 4: Responsive Todo Application
The goal is to create a clean, responsive Todo application that works smoothly across all devices and matches the design provided in the screenshot.
# Features

- Add new tasks
- Delete tasks individually
- Clean and simple UI (replicates the screenshot design)
- Fully responsive layout for mobile, tablet, and desktop
- Uses **localStorage API** to save tasks permanently
- Smooth UI interactions

---

#  Technologies Used

- **HTML5**
- **CSS3**
- **JavaScript (DOM Manipulation)**
- **LocalStorage API**
